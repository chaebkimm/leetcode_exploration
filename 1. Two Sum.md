이 문제를 푸는 가장 간단한 방법은 nums의 모든 조합에 대해 합을 계산해보는 것입니다.

```c++
class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
        for(int i = 1; i < n.size(); ++i)
            for(int j = 0; j < i; ++j)
                if(n[i] + n[j] == target) return {i,j};
        throw;
    }
}
```
